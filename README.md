# m365-entra-scripts
# M365 & Entra ID PowerShell Automation Scripts

A collection of PowerShell scripts for Microsoft 365 and 
Microsoft Entra ID administration, identity management, 
and security operations.

## Scripts

| Script | Description |
|--------|-------------|
| `Get-MFAStatus.ps1` | Export MFA status for all users in tenant |
| `Export-CAPolicy.ps1` | Audit and export all Conditional Access policies |
| `Get-StaleGuests.ps1` | Report guest accounts inactive for 90+ days |
| `Assign-License.ps1` | Bulk license assignment via Microsoft Graph API |
| `Get-MailboxPermissions.ps1` | Export all mailbox delegate permissions |

## Prerequisites
- PowerShell 7+
- Microsoft.Graph PowerShell module
- ExchangeOnlineManagement module
- Entra ID account with appropriate roles
